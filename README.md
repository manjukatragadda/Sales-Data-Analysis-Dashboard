# Sales Data Analysis & Dashboard

## Project Overview
This project analyzes retail sales data and builds an interactive dashboard using Power BI.  
The goal is to identify revenue trends, top-performing products, regional performance, and year-over-year growth insights.

## Objectives
- Clean and preprocess sales data
- Handle missing values and duplicate records
- Analyze monthly sales trends
- Identify top 10 products by revenue
- Compare regional sales performance
- Calculate Year-over-Year (YoY) growth
- Build an interactive Power BI dashboard

## Tools & Technologies
- Power BI
- Python
- Pandas
- Excel / CSV
- DAX

## Dataset
The dataset contains:
- Order ID
- Product Name
- Category
- Region
- Sales Amount
- Order Date
- Quantity

## Data Cleaning
Performed the following preprocessing steps:
- Removed duplicate records
- Handled missing/null values
- Corrected data types
- Formatted date columns
- Removed inconsistent entries

## Dashboard Features
- KPI Cards for Total Sales, Profit, and Orders
- Monthly Revenue Trend Analysis
- Category-wise Sales Analysis
- Top 10 Products by Revenue
- Region-wise Performance
- Year-over-Year Growth Analysis
- Interactive Filters and Slicers

## Key Insights
- Technology category generated the highest revenue
- West region had the best sales performance
- Seasonal sales spikes observed during year-end months
- Top 10 products contributed major share of total revenue
